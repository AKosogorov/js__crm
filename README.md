# CRM системa

## Данное задание было выполнено в качестве курсовой работы по курсу "JavaScript базовый уровень" от компании Skillbox.

Перед запуском убедитесь, что вы установили Node.js версии 12 или выше.

Для запуска сервера перейдите в папку `crm-backend` и выполните команду `node index` через терминал. Для остановки нажмите сочетание клавиш CTRL+C.

После запуска сервера API будет доступен по пути `http://localhost:3000`.

Для запуска приложения перейдите в папку `crm` и запустите `index.html`.


## Цель задания: разработать web-интерфейс для CRM системы, в которой должны быть следующие возможности:

* Просмотр списка людей в виде таблицы
* Добавление нового клиента
* Изменение информации о существующем клиенте
